Lo-fi Chai

This is a website I created for NuCamp coding bootcamp. It displays a venue that plays a niche genre of music called Lo-fi. It displays different areas that customers can enjoy as well as musical peformances from various artists scheduled with food and drink specials for that particular event. This is strictly a front-end website using React.js

Dating Website

This website was created using Udemy tutorials. It allows you to register a user on a fictional dating website and allows various functionality such as "liking" and messaging other users using real-time instant chat features. This is a full-stack application, allowing users to upload photos and receive messages with the assistance of a Postgres Database. This website uses .NET for the back-end and Angular.js for the front-end.
